# Dataset

This project uses the **RAF-DB (Real-world Affective Faces Database)**.

## How to Download
1. Request access at: http://www.whdeng.cn/RAF/model1.html
2. Download and extract the dataset
3. Organize it as:
   data/
   └── RAFDB/
       ├── train/
       │   ├── anger/
       │   ├── disgust/
       │   └── ... (7 classes)
       └── test/
           ├── anger/
           ├── disgust/
           └── ... (7 classes)
